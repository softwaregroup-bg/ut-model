# ut-model version 8 release notes

## Breaking Changes

- Switched from `ut-front-devextreme` to `ut-prime`

## Upgrade Steps

- It is no longer possible to use `ut-front-react` and `ut-model` in the same
  portal. Consider switching the whole UI to `ut-prime`.
